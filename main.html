Projet

Durant mes Deux années de BTS, j'ai eu l'occasion de faire plusieurs projet scolaire et extra, scolaire.

Le premier projet c'est un projet C# nommée gestion materiel, le but de ce projet est de créer une application C# qui à partir de formulaire windows forms, permet d'afficher une liste de nageur, une liste de matériel séparer en 3 tables, avec des héritages, une liste de pret qui sont issue d'une table reliant le materiel au nageur, le tout grâce à une liaison à la base de données.

Le deuxieme projet est un projet laravel, qui permet d'afficher le planning d'une association de nageur avec palme, le but est de pouvoir visualisé les entrainements de l'association il y aura une page de connection, des droits associé à certain entraineur, comme la création de crénaux de cours, un moyen de signalé une indisponnibilié le tout en groupe de 4.
dans cette application nous travaillons avec la méthode Scrum une méthode visant à intéragir souvent avec le client.

Enfin le dernier Projet est un projet Angular (ou vueJs pour d'autre), qui sert à afficher des sondages, créer des sondage, et permet donc de visualisé les réponses au sondage. cette application utilise un web service pour effectuer tout cela, et ce sert de token et d'un systeme de login pour verifier que personne ne répondre deux fois au même sondage.
